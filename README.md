# Bike Buyer Analytics Dashboard

Interactive Excel dashboard analyzing factors that influence bike purchase decisions (income, age, gender, commute distance, etc.)

**Key Business Insights**
- Higher income strongly correlates with bike purchases (especially among males)
- Young buyers (<35) show the strongest purchase intent
- Short commute distances (0–1 miles) have ~2× higher purchase rate

**Technologies**  
Excel • Pivot Tables • Charts • Data Cleaning

## Quick Findings Summary

- Average income of buyers: **$57,963** vs non-buyers **$54,875**
- Top purchasing group: **Young** (232 purchases out of 426)
- Most purchases happen with **0–1 mile** commute

## How to View

1. Open `Bike Sales.xlsx`
2. Go to **Dashboard** sheet
3. Use slicers to filter by region/education/age group

**If Download gives you an HTML file instead** (rare GitHub glitch):
- Right-click the **Raw** button → "Save link as..." → name it `Bike Sales.xlsx`
- Or download the whole repository as ZIP:
  1. Click green **Code** button
  2. Choose **Download ZIP**
  3. Extract → open `Bike Sales.xlsx`

Go to **Dashboard** sheet and use the slicers to explore!
